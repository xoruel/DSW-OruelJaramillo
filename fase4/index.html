Proposito

Fortalecer las habilidades en diseño de sitios web teniendo encuentas las herramientas y técnicas de codificación HTML, CSS y frameworks CSS aplicando técnicas de posicionamiento Web.

Introduccion a las Tematicas

De acuerdo a Prescott, P. (2015), las etiquetas basicas son:

    < !DOCTYPE HTML >
    < html >
    < head >
    < tittle >
    < /head >
    < body >
    < h1 > < /h1 >
    < /body >
    < /html >

Atributos

De acuerdo a Equipo Vertice. (2009), los atributos se pueden definir como:

    "Los atributos son elementos de las etiquetas que pueden proporcionar ciertas caracteristicas a un texto, imagen, etc. (por ejemplo, altura, ancho, color, etc.)".
